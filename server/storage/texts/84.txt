■ 스마트출석관리시스템 통한 출결 관리
강의실에 출석하여 교과목을 수강할 경우, 스마트출석관리시스템을 통해 출결 관리를 합니다. 출석 방법에 따라 다양한 방법으로 편리하고 신속하게 출결 관리가 가능합니다.

- 스마트출석관리시스템 사용 방법
  - 모바일: 구글 플레이스토어/애플 앱스토어에서 "성균관대학교 전자출결" 검색 후 다운로드 → 로그인(킹고포털 ID/비밀번호)
  - PC: https://attend.skku.edu 접속 → 로그인(킹고포털 ID/비밀번호)

- 출석 방법
  - 비콘(Beacon)/강의실에 설치되어 있는 블루투스 장치) 통한 출석
    - (학생) 스마트출석관리시스템 APP 접속 → 비콘 신호체크 → "출석하기" 클릭
  - 출결 단말기 통한 출석
    - (학생) 학생증카드/KINGO-M 신분증(NFC)/KINGO-M QR코드 → 강의실 설치된 출결단말기에 태깅
  - 인증번호 통한 출석
    - (교수) 스마트출석관리시스템 WEB/APP 접속 → 인증번호 생성 → 인증번호 유효시간 설정 → 학생에게 인증번호 안내
    - (학생) 스마트출석관리시스템 APP 접속 → 인증번호 입력
  - 스마트수업부를 통한 출석
    - (교수) 스마트출석관리시스템 WEB/APP 접속 → 적정 출입 → 학생의 출결 상태별 "선택" 버튼 클릭 → [출석, 지각, 결석] 해당 버튼 클릭

- 출결 확인: 스마트출석관리시스템 WEB/APP 접속 → 수업현황 조회 → 출결 상태 확인

※ 출결 상태 설명
  - 출석: 수업 시작 전/종료 전 체크한 경우
  - 지각: 수업 시작 전/종료 전 체크한 경우
  - 결석: 수업 시작 전/종료 전 체크한 경우/교수가 직접 결석 처리한 경우
  - 미인증/체재: 교강사, 학생 모두 시스템으로 출결 처리하지 않은 경우
